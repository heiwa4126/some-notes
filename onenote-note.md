# Microsoft OneNote のメモ

すげえ使いにくいので死ねばいいと思う。
会社で使わされてる人ご愁傷様。

## ページのエクスポート

できない。ページ全体をコピペするのも大変困難。複製はできるのだがcreate atのタイムスタンプが元と同じになる。

## Copilotとの相性

- Copilotペイン真っ白
- 「問題が発生しました」即死
- 入力途中で応答停止

などが高頻度で起きる。使わないにこしたことはない

## ヘッダー

h1, h2 ... に相当するもの。

行選択で

- Ctrl+Alt+1 → 見出し1
- Ctrl+Alt+2 → 見出し2
- Ctrl+Alt+3 → 見出し3

がいちばん簡単。ホームにスタイルはあったりなかったりする。
