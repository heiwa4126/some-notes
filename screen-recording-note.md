# 画面録画 / スクリーンレコーディング(screen recording) メモ

Windows11 で、GUIの操作をビデオにしたい。
動画サイズを小さくしたいので、フレームレートは低い方がいいし、解像度も低い方がいい。
また、画面に変化がないときは、録画しないでほしい。

という話

## OBS Studio がいいらしいので試してみる

[ダウンロード | OBS](https://obsproject.com/ja/download)

1. 「設定」>「ビデオ」で「出力(スケーリング)解像度」を720pに、「FPS共通値」を15-30に変更。
2. 「出力」>「録画」でビットレートを3000kbps前後に（MP4/H.265形式推奨）。
3. 画面変化なし時は「Advanced Scene Switcher」プラグインを追加してモーション検知を設定；静止時は録画を一時停止またはスキップ。
   [Advanced Scene Switcher | OBS Forums](https://obsproject.com/forum/resources/advanced-scene-switcher.395/)

今回セカンドモニタを1024x768にして、そこ全体をキャプチャする前提

最初ウイザードが動く

(左上)シーンの名前を適当に変える「GUI操作録画」(または新規追加)
「GUI操作録画」シーンに、ソース追加(左下)
画面全体を録りたい: 「画面キャプチャ（Display Capture / 画面キャプチャ）」を選択。(今回こっち)
特定アプリだけ: 「ウィンドウキャプチャ」を選択
ソースで「画面キャプチャ」を選ぶと、そのすぐ右に「画面キャプチャ」と歯車(プロパティ)出る。カーソルをキャプチャする

右下の「設定」ボタン
音声→グローバル音声デバイス→全部無効
映像→基本(キャンパス)解像度、出力(スケーリング)解像度を両方とも1024x768
映像→FPSを10に
出力→出力モード「詳細」→録画タブ

- ファイルパス: 適切に設定
- 録画フォーマット: mp4
- エンコーダ: NVIDIA NVENC が使えるならそれを

適用ボタンを押すのを忘れないこと。
