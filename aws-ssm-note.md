# Amazon Systems Manager (SSM) のメモ

AWS Systems Manager
<https://console.aws.amazon.com/systems-manager/home>

## セッションの開始

1. EC2を開始する
2. AWSポータルの AWS Systems Manager へ行き、左ペインの 「ノードツール - セッションマネージャー」へ行く
3. 「セッションの開始」ボタンを押す
4. ターゲットインスタンスでインスタンスIDを入れるなりして、EC2インスタンスを見つける
5. ラジオボタンにチェックして「セッションを開始」ボタンを押す
