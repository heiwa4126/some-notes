# 「検索エンジンの管理」で英語Googleを設定しておくと捗る

Chromeで検索エンジンの管理

`chrome://settings/searchEngines`

を開いて、
その他の検索エンジンの「追加」で

| 項目             | 内容                                     |
| ---------------- | ---------------------------------------- |
| 検索エンジン     | Google(US)                               |
| キーワード       | g                                        |
| URL(%s=検索語句) | https://www.google.com/search?q=%s&hl=en |

のようなのを追加しておくと、
アドレスバーに`g(空白)検索文字列`で英語Googleが検索できるようになって捗る。
