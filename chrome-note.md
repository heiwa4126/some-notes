# Google Chrome のメモ

## Chrome でスクロールして全画面のスクリーンショットをとるには?

(なんか毎回検索してるので Gemini の出力をほぼそのままコピペします)

Google Chrome でスクロールして**全画面のスクリーンショット**(ページ全体)を撮るには、主に**デベロッパーツール**の機能を使用するのが最も確実で拡張機能も不要な方法です。

### 💻 デベロッパーツールを使った方法

以下の手順で操作します。

1.  **デベロッパーツールを開く**

    - スクリーンショットを撮りたい Web ページを開きます。
    - キーボードショートカットを押すか、右クリックメニューから開きます。
      - **Windows/ChromeOS:** $\text{F12}$ キー または $\text{Ctrl} + \text{Shift} + \text{I}$
      - **Mac:** $\text{Command} + \text{Option} + \text{I}$
      - または、ページ内の何もない場所で**右クリック**し、「**検証**」を選択します。

2.  **コマンドメニューを開く**

    - デベロッパーツールが開いたら、以下のショートカットを押して「**実行プロンプト**(Run command)」または「**コマンドメニュー**」を開きます。
      - **Windows/ChromeOS:** $\text{Ctrl} + \text{Shift} + \text{P}$
      - **Mac:** $\text{Command} + \text{Shift} + \text{P}$

3.  **全画面キャプチャのコマンドを実行する**
    - 開いた検索窓に「**screenshot**」または「**full**」と入力します。
    - 候補の中から「**Capture full size screenshot**」を選択してクリック(または $\text{Enter}$ キーを押します)。
    - **ページ全体**のスクリーンショットが自動的に撮影され、PNG ファイルとしてダウンロードフォルダに保存されます。

### 💡 その他の方法(Mac/一部の Windows)

お使いの Chrome のバージョンや OS によっては、以下の方法も使える場合があります。

#### Chrome の設定メニューから

1.  画面右上の「$\vdots$」(縦に点が 3 つ並んだアイコン)をクリックし、「**その他のツール**」にカーソルを合わせます。
2.  表示されたサブメニューに「**スクリーンショット**」という項目があれば、それを選びます。
3.  新しいメニューまたはオプションが表示されたら、「**ページ全体をキャプチャ**」を選択して実行します。

#### エミュレータモードでキャプチャ

特にモバイル表示で全画面を撮りたい場合に便利です。

1.  上記の手順でデベロッパーツールを開きます。
2.  デベロッパーツールの上部にある**デバイスツールバー**のアイコン(スマートフォンとタブレットのアイコン)をクリックして、「**エミュレータモード**」にします。
3.  表示された画面の上部中央付近にある、**デバイス名**の横にある「$\vdots$」(縦に点が 3 つ並んだアイコン)をクリックします。
4.  メニューから「**Capture full size screenshot**」を選択します。

### ⚠️ 注意点

- **最大高さの制限:** Chrome では、非常に長いページの場合、キャプチャできる高さに制限がある場合があります(約 **16,384 ピクセル**など)。
- **ダイナミックコンテンツ:** **無限スクロール**や、スクロールしないと読み込まれないコンテンツがあるページでは、完全なキャプチャができないことがあります。その場合は、事前に一度ページの下部までスクロールしてからキャプチャを試みてください。

これらの方法で、拡張機能を使わずにページ全体を簡単にキャプチャできます。

### "Capture full size screenshot" をショートカットキーに割り振る

この設定は、Chrome ブラウザの機能ではなく、デベロッパーツール(DevTools)自体の設定で行います。

**重要: DevTools のショートカットキーです**

1.  **デベロッパーツールを開く**

    - $\text{F12}$ キー、または $\text{Ctrl} + \text{Shift} + \text{I}$(Windows/ChromeOS)、$\text{Command} + \text{Option} + \text{I}$(Mac)でデベロッパーツールを開きます。

2.  **設定画面を開く**

    - デベロッパーツールの上部にある**歯車アイコン**(⚙️ $\text{Settings}$)をクリックするか、$\text{F1}$ キーを押します。

3.  **ショートカット設定に移動する**

    - 左側のメニューから「**ショートカット** ($\text{Shortcuts}$)」タブを選択します。

4.  **「フルサイズのスクリーンショット」を探す**

    - 設定項目を下にスクロールして、「**Screenshot**」セクションを探します。
    - その中の「**Capture full size screenshot**」という項目を見つけます。

5.  **ショートカットキーを割り当てる**
    - 「Capture full size screenshot」項目の右側にある**鉛筆アイコン**(✏️)をクリックします。
    - 「**ショートカットを追加する** ($\text{Add shortcut}$)」をクリックします。
    - 表示されたテキストボックス内で、**設定したいショートカットキー**(例: $\text{Ctrl} + \text{Q}$ や $\text{Ctrl} + \text{Shift} + \text{F1}$ など、**既存の Chrome ショートカットと重複しないキー**)を実際に押します。
    - キーが入力されたら、**チェックマーク**(✓)をクリックして確定します。

これで、**デベロッパーツールが開いている状態であれば**、設定したショートカットキーを押すだけで、ページ全体のスクリーンショットが撮影されるようになります。
